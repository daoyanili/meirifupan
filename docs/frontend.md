# 前端看板说明

## 页面概述

单页面 HTML 应用，展示涨停复盘数据。深色主题，支持按天查看。

访问地址：http://localhost:8080

## 功能模块

### 1. 顶部导航

- 标题：发家致富 - 涨停复盘
- 日期选择器：切换查看不同交易日数据
- 刷新按钮

### 2. 统计卡片

| 卡片 | 说明 |
|------|------|
| 涨停板块数 | 当日涨停股票所属板块数量 |
| 涨停股数 | 当日涨停股票总数 |
| 热门板块 | 热门板块数量 |
| 首板/连板 | 首板股票数 / 连板股票数 |

### 3. 涨停梯队

按连板数分组展示：
- 最高板在最上面
- 显示股票名称和涨停时间
- 标注每个梯队的股票数量

### 4. 热门板块 Top 10

- 板块名称和热度分数
- 进度条可视化（相对最高分）

### 5. 涨停股明细表格

| 列 | 说明 |
|------|------|
| 股票 | 股票名称 |
| 代码 | 股票代码 |
| 板块 | 所属板块 |
| 涨停时间 | 封板时间 |
| 连板 | 首板/2板/3板... |
| 涨停原因 | 简要原因（hover 查看完整） |

支持按板块筛选。

## 数据加载

```javascript
// 从 JSON 文件加载数据
fetch(`data/uplimit/uplimit_${date}.json`)
```

## 可用日期

在 `index.html` 的 `dates` 数组中配置：
```javascript
const dates = [
    '2026-06-01', '2026-05-28', '2026-05-27', ...
];
```

添加新日期后需手动更新此数组。
